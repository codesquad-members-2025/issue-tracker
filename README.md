## 브랜치 구조 👀

```
main
├── develop-BE
│   ├── feature/signup
│   ├── feature/comment
│   └── ...
├── develop-FE
│   ├── feature/signup
│   ├── feature/comment
│   └── ...
```

- main: 배포 가능한 코드
- develop: 통합 개발 브랜치
- feature/\*: 기능 개발 브랜치
- refactor/\*: 리팩토링 브랜치
- fix/\*: 디버깅 브랜치
